# homebrew-tap

Homebrew formulae for EIT UMD tools.

## Usage

```sh
brew tap eitumd/tap
```

## Available formulae

| Formula | Description |
|---------|-------------|
| `aws-oidc` | OIDC credential helper for AWS CLI using PKCE |
| `eit-cli` | CLI tool for EIT services (Statuspage, JitBit, LDAP, BeyondTrust, and more) |

```sh
brew install aws-oidc
brew install eit-cli
```
